
# 小技巧
## 1.生成二维数组

1. **`Array.from()`** 方法有一个可选参数 **`mapFn`**，可以在最后生成的数组上再执行一次 **map**方法后再返回。

```js
const m = 3, n=4;
let arr = Array.from(Array(m),()=>Array(n));
```

2. `new Array(n).fill(0).map(()=> Array(m).fill(0)` 使用map也可以快速创建，但是需要保证初始的一维数组内容不是`undefined`不然也会因没有参数出错。
