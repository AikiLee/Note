## 基础概念

任何以`.py`结尾的python文件都会自动被看作一个模块例如，`utils.py` 就是一个名为 `utils`的模块。

使用python模块模式启动项目
## 导入

三种方式

1. `import xxx`导入一个模块中所有方法
2. `from xxx import xxx` 从指定模块中导入指定方法
3. `import xxx as alias` 给导入的模块重命名
4. `from xxx import *` 导入一个模块中的所有函数和变量

## 包

当部分文件放在同一个文件夹下，可以称为一个包，例如

```shell
app
--> a.py
--> b.py
--> c.py

使用方式：
from app.a import xxx 
```

一个文件夹要被识别为包，需要包含一个文件`__init__.py`(3.3以前)
